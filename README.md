# Proyecto_Fundamento_bases_de_datos

# Nombre del proyecto:
Inventario Personal Historico

# Descripcion:
En este proyecto podras hacer una lista con los objetos que tienes independientemente cuales sean, Creas una categoria personalizada dentro de la cual podras agregar objetos, puedes crear mas categorias diferentes para objetos distintos. Deespues Creas/agregas el objeto a la categoria correspondiente. Tambien habra una categoria digamosle ya definida, que seria como "una lista de deseos" en la que agregarias manualmente algun producto/objeto que no tengas, pero que te gustaria adquirir en un futuro. A su vez tambien se agregara una seccion en la que iran los objetos que aparecen en el inventario, pero tiempo despues por x o y razon ya no los tienes y aparezcan asi, como objetos que ya no tienes pero para no olvidarlos si asi se podria decir

# Motivacion:
Quiza sea tan facil como simplemente anotarlos en una hoja o algun excell, pero (almenos a mi) se me olvida que tengo algunas cosas y considero que tener un registro de cosas que tengo y si algun dia pierdo, me deshago de cierto objeto tambien poder registrar eso y que no quede como un objetro mas y pudiera llegar a recordar esporadicamente. Basicamente mi motivacion es el deseo de preservar un recuerdo, Si aun tengo algo, recordar donde lo deje  y si ya no lo tengo axctualizar la lista y tambien como llevar un registro de que o cuantos objetos adquiero a lo largo de mi vida y ayudarme a evaluar ciertas cosas.
